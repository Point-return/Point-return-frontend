# Point-return-frontend

## User Interface of the recommendation system Point return. 
[Point-return](https://point-return.github.io)

## Install
 - node 20.9.0+
 - npm 10.1.0+

## Getting started

```
    npm run ci
    npm run dev
    npm run build
```
